# Sistema de Adopción de Mascotas

Un sistema diseñado para facilitar el proceso de adopción de mascotas, ofreciendo una plataforma donde los usuarios pueden explorar perfiles de mascotas disponibles y realizar solicitudes de adopción.

### Detalles

- **Lenguaje principal:** HTML (72.7% del proyecto).  
- **Diseño:** Construido con CSS para una interfaz visual limpia y atractiva.  
- **Objetivo:** Brindar una solución práctica y accesible para conectar adoptantes con refugios o dueños de mascotas.

### Características principales

- Listado de perfiles de mascotas disponibles para adopción.  
- Información detallada de cada mascota.  
- Interfaz responsiva, adecuada para dispositivos móviles y de escritorio.  
- Diseño intuitivo para facilitar la navegación.
